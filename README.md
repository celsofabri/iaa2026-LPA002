# Projeto de Análise de Carros no Brasil

Este repositório contém os resultados de uma análise de dados sobre preços de carros no Brasil, incluindo visualizações e um `index.html` para fácil visualização.

## Visualizar no GitHub Pages

Você pode acessar o site do projeto através do GitHub Pages neste link: `[Link do GitHub Pages será inserido aqui após a ativação]`

## Conteúdo do Projeto

- `index.html`: Página principal com informações sobre o projeto e links para os gráficos.
- `precos_carros_brasil.csv`: Conjunto de dados utilizado na análise.
- `trabalho_lpa_notebook.ipynb`: Notebook Jupyter com a análise completa.

### Gráficos Gerados

- `grafico_2a_carros_por_marca.png`: Distribuição de carros por marca.
- `grafico_2b_carros_por_engrenagem.png`: Distribuição de carros por tipo de engrenagem.
- `grafico_2c_evolucao_preco_mensal.png`: Evolução do preço médio mensal.
- `grafico_2d_preco_marca_engrenagem.png`: Preço por marca e tipo de engrenagem.
- `grafico_2f_preco_marca_combustivel.png`: Preço por marca e tipo de combustível.
- `grafico_3e_comparacao_importancia.png`: Comparação da importância de features.
- `grafico_3e_importancia_rf.png`: Importância de features (Random Forest).
- `grafico_3e_importancia_xgb.png`: Importância de features (XGBoost).
- `grafico_3g_comparacao_metricas.png`: Comparação de métricas de modelo.
- `grafico_3g_real_vs_predito.png`: Valores reais vs. preditos.
